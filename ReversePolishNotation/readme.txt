Задание на лабораторную:
Написать ПО со следующей функциональностью:
1) ввод функции в текстовом формате (например, "-2*x + sin(0.5*x)"),
2) ввод границ отрезка и шага аргумента (например, a = 1, b = 3.14, h
= 0.5),
3) вывод таблицы значений функции во всех точках, включая границы
отрезка.
Ввод-вывод можно сделать консольный, можно файловый, можно разработать
графический интерфейс – по желанию разработчика.
Язык программирования, как всегда, любой.
Реализовать 5 указанных операций (+,-,*,/,sin), другие - по желанию.
Обязательно проверять ошибки, обозначенные в конспекте, как на этапе
перевода, так и на этапе вычисления (деление на ноль).
